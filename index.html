Trust stack

Where CiteSafe pulls truth

Independent of the model that wrote the brief. Named risk layers + public primary indexes. We verify citation existence and fabrication risk — not KeyCite / Shepard's treatment.

Fabrication heuristics

Impossible years, fake reporters, invented party names — flagged before filing.

CiteSafe authority corpus

Curated high-signal known authorities. Labeled corpus — never sold as a full library.

CourtListener

Free Law Project public case index for opinion existence checks (API).

Caselaw Access Project

Harvard CAP case index for secondary existence signals (API).

GovInfo / statutes

Public catalog signals for U.S.C.-style forms — not full statutory research.

Pipeline

Extract → risk heuristics → corpus → CourtListener / CAP / GovInfo → status export with source badges and report hash.
