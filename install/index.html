iOS — Download now

iPhone & iPad · Safari only · Beta

  1. Tap Download now on iOS above (opens CiteSafe in Safari)
  2. Tap the Share button (□ with ↑)
  3. Scroll and tap Add to Home Screen
  4. Tap Add — blue CiteSafe icon appears

Apple blocks automatic install. Share → Add to Home Screen is the only way (beta).

Android — Download now

Android · Chrome · Beta

  1. Tap Download now on Android
  2. If a popup appears, tap Install
  3. Or menu ⋮ → Install app / Add to Home screen
  4. Open CiteSafe from your home screen

After install

Open the home-screen icon. You get the full app (scan, dashboard, profile). Real App Store / Play listings come later.
