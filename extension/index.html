How to add CiteSafe to Chrome

Free beta. One time setup. About two minutes. Chrome Web Store listing comes after Google review. This install works today.

Step 0 · Download citesafe-chrome-extension.zip
  1. 1
    Unzip the file

    Right click the zip → Extract All. Keep that folder. Chrome needs it to stay on your computer.

  2. 2
    Open chrome://extensions

    In Chrome, click the address bar, type chrome://extensions and press Enter. Or menu ⋮ → Extensions → Manage extensions.

  3. 3
    Turn on Developer mode

    Top right of that page. Flip Developer mode ON.

  4. 4
    Load unpacked

    Click Load unpacked. Select the unzipped folder (the one that contains manifest.json). Click Select Folder.

  5. 5
    Pin CiteSafe

    Puzzle icon top right of Chrome → pin CiteSafe so it stays on your toolbar.

  6. 6
    Use it before production

    Click CiteSafe → paste your draft or hit Sample. Or highlight text on any page → right click → CiteSafe: check citations before production.
